# Java代理模式

## 代理模式分类:

+ 静态代理
+ 动态代理

### 代理模式的好处

+ 可以使真实角色的操作更加纯粹!不用去关注一些公共业务
+ 公共也可以交给代理角色!实现了业务的分工!
+ 公共业务发生扩展的时候,方便集中管理

代理模式的缺点:

+ 一个真实的角色就会产生一个代理角色;代码量会翻倍-开发效率会降低

#### 动态代理

+ 动态代理和静态代理角色一样
+ 动态代理的代理类是动态生成的,不是我们直接写好的
+ 动态代理分为两大类:
  + 基于接口的动态代理
    + jdk 动态代理
  + 基于类的动态代理
    + cglib(一个类)
    + java字节码实现: (用的多的)javasist
